# RAIV for mac v0.3.0-alpha

RAIV for mac is a free, open-source comic and image viewer for Apple Silicon
Macs. This alpha bundles the application, Real-CUGAN engine, and AI models.
Python, uv, and a separate engine installation are not required.

## Highlights

- English and Japanese interface with automatic macOS language detection
- Language selector in the bookshelf header
- Local bookshelf for CBZ, CBR, ZIP, RAR, 7z, image folders, and images
- Right-to-left two-page reading for manga
- Automatic Real-CUGAN enhancement with original-image comparison
- Faster back-and-forth navigation and bounded forward/backward prefetch caches
- Freely licensed Pepper&Carrot demo books and screenshots in the repository

## Installation

1. Download the standalone ZIP and verify its optional SHA-256 checksum.
2. Extract `RAIV.app` and move it to Applications.
3. On first launch, Control-click `RAIV.app` and choose **Open**.

This alpha is ad-hoc signed but is not Apple Developer ID signed or notarized.
It supports Apple Silicon Macs; Intel Macs are not currently supported.

See [INSTALL.md](../../INSTALL.md) for details.
