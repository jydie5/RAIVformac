# Free demo books

Drop any ZIP in this directory onto the RAIV bookshelf.

The archives contain freely redistributable *Pepper&Carrot* pages so users and
contributors can test archive import, cover generation, natural ordering,
two-page display, and AI enhancement without supplying copyrighted commercial
manga.

See [ATTRIBUTION.md](ATTRIBUTION.md) for the author, source, and CC BY 4.0
license. The same attribution file is embedded in every ZIP.
